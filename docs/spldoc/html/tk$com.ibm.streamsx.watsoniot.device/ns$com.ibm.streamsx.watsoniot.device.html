Namespace com.ibm.streamsx.watsoniot.device

Toolkits > com.ibm.streamsx.watsoniot.device 1.0.0 > com.ibm.streamsx.watsoniot.device

Operators

  • WatsonIoTDeviceConnector: The WatsonIoTDeviceConnector operator connects an SPL graph to the Watson IoT Platform as an IoT 'device': it encodes input tuples as 'events' and sends them to IoT applications; concurrently, it recieves 'commands' from IoT applications and decodes them into output tuples.
  • WatsonIoTDeviceSink: The WatsonIoTDeviceSink operator connects an SPL graph to the Watson IoT Platform as an IoT 'device': it encodes input tuples as 'events' and sends them to IoT applications.
  • WatsonIoTDeviceSource: The WatsonIoTDeviceSource operator connects an SPL graph to the Watson IoT Platform as an IoT 'device': it recieves 'commands' from IoT devices and decodes them into output tuples.
